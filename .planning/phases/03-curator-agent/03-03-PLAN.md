<objective>
Build the `/curate-bank` Claude skill that the user invokes with natural language to add or remove questions in the daily-learning bank. The skill reuses the `daily-question-curator` quality bar (does not duplicate it), writes to questions.md as the source of truth, then syncs the change to the Google Sheet via `scripts/sheets_helper.py`. A preview-and-confirm gate sits in front of every write. A divergence check protects against silently overwriting manual sheet edits. Removal matching is fuzzy with explicit confirmation on multiple candidates.

Purpose: This is the visible product surface of Phase 3 — without it, the helper and credential built in Plans 01 and 02 are inert plumbing. FUTURE-01 says "Claude agent that edits the Bank tab (add/remove questions based on user input)"; this plan delivers exactly that, wrapping the existing curator skill so the quality bar stays in one place.

Output: One SKILL.md file at `.claude/skills/curate-bank/SKILL.md` containing the skill definition, plus an end-to-end human-verify checkpoint exercising one add and one remove against the live bank.
</objective>

| skill output → user | Preview rendering must never include the answer to a question |
