CLI contract for scripts/sheets_helper.py:

  python scripts/sheets_helper.py read
    → stdout: JSON array of row objects, one per data row in the Bank tab
      [{"row": 2, "question": "...", "domain": "...", "est_minutes": "15", "status": "queued", "date_sent": ""}, ...]
    → exit 0 on success, non-zero on auth/API error with error JSON on stderr

  python scripts/sheets_helper.py append --question "..." --domain "..." --est-minutes 15
    → appends one row with status=queued, date_sent=""
    → stdout: JSON {"appended_row": <new_row_index>, "question": "..."}
    → exit 0 on success

  python scripts/sheets_helper.py delete --question "<exact question text>"
    → finds row whose question column matches the argument exactly, deletes it
    → stdout: JSON {"deleted_row": <row_index>, "question": "..."}
    → exit 2 if no match found; exit 3 if multiple matches found (returns candidate rows on stderr)
    → exit 0 on single successful match + delete

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| local repo → Google Sheets API | Service account JWT crosses local→cloud; credential file is the trust anchor |
| user input → helper CLI args | Loose strings supplied by the user via the skill flow are interpolated into Sheets API row payloads |
| filesystem → git history | `.secrets/` contents must never enter git history |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Information disclosure | .secrets/service-account.json | mitigate | `.gitignore` line `.secrets/` blocks accidental commit; Task 1 acceptance verifies via `git check-ignore` that the path is actually ignored |
| T-03-02 | Tampering | scripts/sheets_helper.py append/delete | mitigate | All Sheets API mutations require the service account credential; the credential file is local-only and not bundled with the script |
| T-03-03 | Repudiation | helper deletes wrong row due to ambiguous match | mitigate | `delete` exits with code 3 and lists all candidate row indices on stderr when >1 match, never silently picks one |
| T-03-04 | Denial of service | malformed CLI args crash helper | accept | Helper is invoked only by the local skill on the user's own machine; argparse + top-level try/except convert errors into non-zero exits with JSON stderr |
| T-03-05 | Elevation of privilege | service account scope creep | mitigate | SCOPES constant is fixed to `https://www.googleapis.com/auth/spreadsheets`; broader scopes would require source change + redeploy |
| T-03-SC | Tampering (supply chain) | pip installs (google-auth, google-api-python-client) | mitigate | Versions pinned in scripts/requirements.txt (`google-auth==2.35.0`, `google-auth-httplib2==0.2.0`, `google-api-python-client==2.149.0`); these are first-party Google packages on PyPI (legitimacy: official). No third-party libraries introduced in this plan. |
</threat_model>
